# Mapeamento Inteligente de Processos com IA

Projeto acadêmico para captura de eventos de interação do usuário, reconstrução automática de processos e geração automática de documentação, incluindo fluxogramas, descrições textuais e POPs.

## Tecnologias
- Python
- Git/GitHub

## Estrutura inicial
- Captura de eventos
- Processamento e análise
- Reconstrução de fluxo
- Geração de documentação
